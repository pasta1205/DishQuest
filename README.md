## 📌 About

**DishQuest** is a modern recipe-finding web application that helps users discover dishes based on ingredients or preferences. It provides an intuitive interface, quick search functionality, and AI-powered suggestions to make cooking easier and more enjoyable.

---

## ✨ Features

- 🔍 Search recipes by name  
- 🥕 Find recipes using available ingredients  
- 🎲 Generate random recipes instantly  
- 📖 View detailed recipe instructions in a modal  
- 💬 Recipies suggestions using MEALDB API  
- 🎨 Clean, responsive, and user-friendly UI  

---

## 🛠️ Tech Stack

- **HTML5** – Page structure  
- **CSS3** – Styling & responsive design  
- **JavaScript (Vanilla JS)** – Functionality  
- **MEAL DB** – API
